Acerca de este sitio web

Este sitio web fue creado en colaboración con el Trabajo Comunal Universitario TC-691 "Tropicalización de la Tecnología", coordinado desde la Escuela de Ingeniería Eléctrica de la Universidad de Costa Rica.
#SomosAcciónSocialUCR

Tecnologías

Django
Framework en Python para el back-end.
Bootstrap
Framework de HTML, CSS y JavaScript para el front-end.
Vue.js
Framework en JavaScript para aplicaciones (widgets) en el front-end.

Este es un proyecto de software libre disponible en GitHub.
